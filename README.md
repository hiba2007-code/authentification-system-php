# PHP Authentication System

## Description
This project is a simple authentication system built with PHP and MySQL. 
It allows users to register, login, and logout securely.

## Features
- User registration
- User login
- Password hashing
- Session management
- Logout system

## Technologies
- PHP
- MySQL
- HTML/CSS

## Installation
1. Clone the repository:
git clone https://github.com/hiba2007-code/authentification-system-php.git

2. Move the project to htdocs (XAMPP)

3. Import the database (database.sql) into phpMyAdmin

4. Start Apache and MySQL

5. Open in browser:
http://localhost/authentification-system-php

## Database
Make sure to create a database and import the SQL file included in the project.
